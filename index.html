LearningConn

LearningConn is an application for students with learning disabilities that focuses on giving the power of accommodations to the student so they can understand what they need and properly communicate that with the teacher.

  • Xcode
  • Swift

Why: This project I started in high school and still work on today as I take classes that pertain to it. The main goal of this project was to come up with an idea to help younger students learn how to advocate for themselves in the classroom.

Real-Time Rowing

During my time at Northeastern I engaged in research where I looked at real time feedback systems and how they pertain to sports. Specifically I looked at how real-time feedback would pertain to rowing. Using two accelerometers and a local LLM I was able to create an application that gave users real-time feedback in the matter of seconds, and during each stroke.

  • Python
  • Web Development
  • Accelerometer Data

Why: I was interested in this topic because it would mean making coaching more accessible for everyone. High-level coaching is hard to come by and makes it so that those who have it have an unfair advantage, so I thought it would be best to find a way to make it more accessible.

Training Tracker

This application is meant for rowers to have the data they need. I have noticed that, especially in endurance sports, data is becoming king. While data is important, it is not the end-all-be-all — it should be taken with a grain of salt. So I created an application that allows rowers to see the data that matters and not get overwhelmed with everything else.

  • Xcode
  • Swift

Why: I wanted to help rowers see that not all data is good data. You only need the data that is going to help you physically and mentally.
